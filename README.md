🛰️ Artemis Hub

# Artemis Hub — це багатофункціональна оболонка (OS Shell) для швидкого доступу до системних інструментів та космічних даних.

# ✨ Основні можливості
- 🌌 **Space Weather:** Отримання актуальних фото та даних від NASA API.
- 🌍 **Translator:** Вбудований перекладач (UA/EN).
- 📊 **System Report:** Моніторинг ресурсів системи (CPU, RAM).
- 🛠️ **Utility Kit:** QR-генератор, калькулятор, блокнот та малювання.
- 🌐 **Localization:** Повна підтримка української та англійської мов.

# 🛠️ Технології
- **Language:** Python 3.x
- **GUI:** Tkinter / CustomTkinter
- **Image Processing:** Pillow (PIL)
- **API:** NASA Open API, GoogleTrans
- **Version Control:** Git & GitHub

# 🚀 Інсталяція
1. Клонуйте репозиторій:
   `git clone https://github.com/SigmaBudanoff/ArtemisHub.git`
2. Встановіть залежності:
   `pip install pillow requests googletrans==3.1.0a0 psutil`
3. Запустіть головний файл:
   `python Artemis_OS.py`
